Usage:
    npx common-frontend build [options]
    npm run <script> -- build [options]

Options:
    --config, -c <file path> ........... file path to the build config file
    --development, -dev, -d ............ enables development mode
    --no-clean ......................... dont clean/delete build directory before builds
    --notify, -n [title] ............... enables non cli notifications,
                                         > optional title can be provided
    --process .......................... enables processing regardless of build context
                                         > processing disabled for dev builds by default
                                           this setting can be used to force it to run
    --watch, -w  ....................... enables watching for changes and auto rebuilds
                                         > if a proxy url is provided a browser sync
                                           instance will be started
    --watch-proxy <proxy url> .......... the url of a site to proxy with browser sync
                                         during watch, only taken into account if
                                         -w|--watch argument exists
    --watch-proxy-delay <seconds> ...... the delay prior to update injection when a
                                         watch proxy is being used, only taken into
                                         account if --watch-proxy argument exists
